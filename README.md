# Paddock Pulse — F1 Management System

A responsive, multi-page Formula 1 fan-hub website built with HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

## Live Pages
- `index.html` — Home: hero, live race countdown, championship snapshot
- `about.html` — Project background and 2026 regulation changes
- `drivers.html` — Full grid with team filter
- `teams.html` — All 11 constructors with status filter
- `schedule.html` — Full 2026 race calendar
- `news.html` — Season storylines and headlines
- `gallery.html` — Filterable race-day gallery
- `contact.html` — Contact form with validation + FAQ accordion

## Features
- Dark/light theme toggle (persisted via localStorage)
- Live countdown timer to the next race
- Filterable driver, team, and gallery grids
- Contact form with custom JavaScript validation
- Fully responsive (Bootstrap grid + custom breakpoints)

## Tech Stack
HTML5, CSS3 (custom properties / design tokens), JavaScript (ES6+), Bootstrap 5.3
